SPC 공정능력 분석 — 타 PC 배포 안내 (IT / 담당자용)
================================================

[배포물]
  release 폴더의 ZIP 파일 1개
  예: SPC_공정능력분석_v1.0.0_Win64_20260604.zip

[대상 환경]
  · Windows 10 / 11 (64bit)
  · Python 설치 불필요
  · 인터넷 불필요 (오프라인 실행 가능)

[설치 절차 — 사용자 PC]
  1. ZIP을 로컬 디스크에 압축 해제
     권장: C:\Program Files\SPC_공정능력분석\
           또는 D:\Tools\SPC_공정능력분석\
     ※ 네트워크 드라이브(Z:)만에 두면 느리거나 차단될 수 있음

  2. 압축 해제한 폴더 안 구조 확인
     · SPC_공정능력분석.exe
     · _internal\  (필수, 삭제 금지)
     · data\input\ , data\output\
     · 실행_SPC.bat , 사용방법.txt

  3. (선택) 바탕화면_바로가기_만들기.bat 실행

  4. 첫 실행
     · 「실행_SPC.bat」 또는 exe 더블클릭
     · Windows Defender / SmartScreen 경고 시
       「추가 정보」→「실행」 또는 관리자가 예외 등록
     · ZIP에서 바로 실행하지 말고, 반드시 압축 해제 후 실행

[보안 / 백신]
  · PyInstaller exe는 신규 빌드 시 백신이 오탐할 수 있음
  · 사내 백신 예외: 압축 해제 폴더 전체 경로 등록 권장
  · 필요 시 폴더 우클릭 → 속성 → 「차단 해제」 체크 (Windows)

[업데이트]
  · 기존 폴더를 삭제하거나 이름 변경 후 새 ZIP 압축 해제
  · data\output\ 결과만 백업 후 교체 가능

[문의 시 수집 정보]
  · Windows 버전
  · 오류 메시지 스크린샷 (「SPC 시작 오류」 창 포함)
  · MES 파일 형식 (xlsx / 암호화 여부)
